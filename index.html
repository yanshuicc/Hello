这个是github基本功能的记录。

系统为ubuntu。

git安装比较简单

sudo apt-get install git

然后初始化

git config --global user.name "Your Name"
git config --global user.email "email@example.com"

生成密钥,~/.ssh/中有d_rsa和id_rsa.pub两个文件，id_rsa是私钥，不能泄露出去，id_rsa.pub是公钥，可以告诉任何人

ssh-keygen -t rsa -C "1258474820@qq.com"

登陆GitHub，打开“Account settings”，“SSH Keys”页面：

然后，点“Add SSH Key”，填上任意Title，在Key文本框里粘贴id_rsa.pub文件的内容

远程库和本地库绑定，origin为远程库的名字

git remote add origin git@github.com:yanshuicc/Hello.git

远程库推送，第一次推送-u绑定远程分支和本地分支

git push [-u] origin master

创建版本库

git init

提交文件，commit只提交add的文件

git add <file-name>
git commit -m ""

查看repository状况

git status

查看修改变化

git diff

查看最近提交记录

git log [--pretty=oneline]

撤销修改,撤销未add文件的修改，回到上次add的状态。

git checkout -- <file-name>

撤销add,撤销未commit文件的修改，回到上次commit的状态。

git reset HEAD <file-name>

恢复版本，HEAD表示当前版本，^表示上个，获取前面Ｎ个commit版本。

git reset --hard HEAD[^|~N]
git reset --hard <版本号>

查看操作历史命令

git reflog

删除文件

rm <file-name>
git rm <file-name>
git commit -m ""

分支

创建分支

git branch <brach-name>

切换分支

git checkout <branch-name>

创建并切换分支

git checkout -b <branch-name>

查看分支

git branch

合并指定分支到当前分支

git merge <branch-name>

删除分支

git branch -d <name>
